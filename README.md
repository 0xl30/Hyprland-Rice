# Hyprland-Rice 🚀

Welcome to my Hyprland Rice setup! This is not entirely my own work; I have added some custom code and features to make it perfect for my needs.

## 🎨 Extra Features of this Rice:
- 🌈 **Fully Rainbow Borders** on notifications, Waybar windows, and more.
- 🔒 **New Waybar Modules** for VPN, Keyboard Input Changer, Cava (with rainbow color effects), and more.
- 🎨 **New Rofi Menu, Notification Panel, Music Player Menu** for an enhanced user experience.
- 💠 **New Bluesh Themes** for icons, cursor, and fonts.
- 🎨 **ZSH Customization**: New themes and plugins for ZSH.
- 📝 **Profile + Keybind Hints Page**: Easier navigation and control.
- 🎶 **yt-dlp Cookie Functions**: Seamless music playback when connected to a VPN (no errors, no reCAPTCHA verification required).
